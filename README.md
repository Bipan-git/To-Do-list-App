# To Do List App

A Simple and responsive **React.js** To Do list Application that allows users to **add,edit,delete , mark complete ** and ** filter tasks** .
All tasks are saved using **Local Storage** , so they stay even after refreshin the page !

---

## Features

- ✅Add new tasks
- ✅Edit existing tasks
- ✅Delete tasks
- ✅mark tasks as completed / pending
- ✅filter tasks (All/completed/pending)
- ✅Responsive desing (works on mobile )
- ✅saves data in Local Storage

---

## Technologies used

- React.js(vite) - for building the user interface
- HTML5 & CSS3 -for logic and interactivity
- Local Storage API -to persist data
